四、每个系统最值得学走的设计

系统 最值得学走的设计 一句话
self-improving-skills Nudge-once + fail-safe hook “用 transcript JSONL 的实际结构做判断，任何异常都 fallback 到 approve”
EvoSkill Feedback Descent 算法 “失败时累积反馈，成功时清空历史——133 行代码”
SkillRL 分层 SKILLBANK + 动态更新 “general skills + task-specific skills + mistakes，LLM 分析失败→生成新技能”
ouroboros 宪法治理 + 后台意识 “Agent 不只是工具，是有身份、有记忆、会主动思考的数字存在”
evolver Gene + Capsule 分离 “假设和实验证据分开存储，表观标记控制基因表达”
gbase 三层认知沉淀 “经验(what)→认知(why)→策略(how)，逐层升维抽象”
hermes-agent Footprint Ladder “核心窄，能力在边缘——每个新增 core tool 都是成本”
quant cluster 交易学习闭环 “每笔交易→学习记录→replay→归因→promote only if better”
